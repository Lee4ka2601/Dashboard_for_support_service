### Дашборд для Английский от Практикум
**Источник данных** - 
Данные с метриками службы поддержки Яндекс Практикума в Google Sheets за апрель-май 2024 г.


**Цель исследования** — мне необходимо перевести текущие данные из Google Sheets в DataLens для службы поддержки Яндекс Практикума, построить дашборд.


**Ход исследования**

1. Перенос, обзор  данных, добавление расчетных полей в датасете; 
 
2. Создание чартов согласно таблицам Заказчика;
 
3. Создание дашборда, настройка селекторов, параметров для выбора нужных показателей и вывода метрик на экран;
 
4. Сформулировать/ добавить предложения по улучшению на дашборд.
 

**Используемые навыки и инструменты**

   Google Sheets, DataLens.

**Общий вывод**

Данные из Google Sheets за апрель-май 2024 г. перенесены в  DataLens, дашборд построен.
